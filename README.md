# Pharmacy Performance Dashboard

A Power BI dashboard analyzing pharmacy operations, drug movement, patient demographics, and workflow efficiency across two independent pharmacies.

## Dataset
pharmacy_operations_dataset.csv

## Dashboard Pages
1. Pharmacy Performance Overview
2. Inventory & Drug Movement
3. Workflow & Efficiency Insights

## Tools Used
- Power BI
- Power Query
- DAX
- CSV dataset

## Key Insights
- Top-performing drugs
- High-demand categories
- Low-stock alerting
- Digital vs walk-in ordering
- Waiting & processing time trends
